Monthly Listing Newsletter

How It Works
1
One --listing flag per property — pipe-delimited: "Address | Price | Specs | Photo URL". Price, specs, and photo are optional but the more you give it, the better the copy and email card look.
2
Claude writes the copy — it sees your price and specs and generates an ALL-CAPS headline, 2–3 sentence investment pitch, and a CTA. Commercial tone aimed at developers and investors.
3
Each listing becomes a photo card — full-width hero photo, dark header with headline, price badge, spec chips (size · zoning · highlights), AI copy, and a contact button. Photos load from any URL — LoopNet, your own hosted images, anywhere.
4
Always previews to you first — email goes to arecblackhills@gmail.com with [PREVIEW] tag. Review it in Gmail, then add --send-all to deliver to the full list.
Claude AI — Copy Generation
The agent calls the Anthropic API (Claude claude-sonnet-4-6) to write marketing copy for each property you list. You need to set the ANTHROPIC_API_KEY environment variable before running:

export ANTHROPIC_API_KEY="your-key-here"

If the key is not set, the agent falls back to plain property addresses in the email (no AI copy). Get your key at console.anthropic.com.
How to Send
Run from: ~/arc-dashboard/agents/listing_newsletter/
Command What happens
python3 generate_newsletter.py --properties "addr1, addr2" --dry-run No emails — saves preview_listing_newsletter.html you can open in your browser to review the full layout
python3 generate_newsletter.py --properties "addr1, addr2" Sends only to arecblackhills@gmail.com with [PREVIEW] tag — check it in Gmail before sending to the list
python3 generate_newsletter.py --properties "addr1, addr2" --send-all Preview to Kevin first, then delivers to all 32 recipients
python3 generate_newsletter.py --properties "..." --message "..." --send-all Adds a custom intro paragraph at the top of the email before the property listings
Example — This Month
Sample Command
python3 generate_newsletter.py \
--listing "Highway 1416 Box Elder SD | $3,750,000 | 24.75 AC · C-2 · Hwy Frontage | https://images1.loopnet.com/i2/QGILchjq.../photo.png" \
--listing "456 Tower Rd Rapid City SD | $1,200,000 | 8 AC · Multifamily Ready" \
--message "Strong demand heading into Q3. Ellsworth corridor continues to attract institutional capital." \
--subject "Black Hills Commercial — July 2026 Featured Listings"
One --listing per property. Each field separated by |: Name, Price, Specs, Photo URL. Photo URL is optional — paste the LoopNet image URL directly. Claude generates the headline and copy from your specs. Add --dry-run first, --send-all when ready.
Dry Run — See It Before Sending
python3 generate_newsletter.py \
--listing "Highway 1416 Box Elder SD | $3,750,000 | 24.75 AC · C-2 | https://your-photo-url.jpg" \
--dry-run
Saves preview_listing_newsletter.html in the same folder. Open it in any browser — shows the exact email with photo card, price badge, spec chips, and AI-generated copy. No emails sent.

One-Time Setup
1
Get an Anthropic API key — go to console.anthropic.com, create an account, and copy your API key. Add it to your shell: export ANTHROPIC_API_KEY="sk-ant-..."
2
Install dependencies (one time) — pip3 install anthropic
3
Test with dry run — run the sample command above with --dry-run, open the preview HTML, confirm it looks right.
4
Send to yourself first — run without --send-all to get the preview email at arecblackhills@gmail.com, review in Gmail, then add --send-all to go live.
