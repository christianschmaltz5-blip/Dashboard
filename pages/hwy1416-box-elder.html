How to Send This Newsletter
Run from: ~/arc-dashboard/agents/hwy1416_newsletter/
Command What happens
python3 send_newsletter.py --dry-run No emails sent — saves a preview HTML file you can open in your browser
python3 send_newsletter.py Sends only to arecblackhills@gmail.com with [PREVIEW] tag — review before sending to anyone else
python3 send_newsletter.py --send-all Sends preview to Kevin first, then delivers to all 32 recipients with PDF attached
python3 send_newsletter.py --message "..." --send-all Custom update text this week, then full send — use \n\n between paragraphs

Email template — Design the recurring email layout with project photos, update text, and contact info. Kevin to provide development photos and key talking points for the first send.
2
First send — Review recipient list, confirm all addresses, and send the first update manually to validate deliverability before scheduling automatic sends.
3
Scheduler — Register a launchd job alongside the market report to automate weekly sends. Weekly day/time TBD with Kevin.
4
Content updates — Each week Kevin provides new photos or a one-paragraph update. The agent formats it into the email template and sends automatically.
